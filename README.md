# Library Project

> [!NOTE]
> Этот репозиторий — учебный Django-проект для автоматизации базовых библиотечных операций: учёта книг, читателей, выдач, возвратов и контроля просрочек.

> - `library_project/library/templates/` — HTML-шаблоны интерфейса;
> - `library_project/library_project/settings.py` — конфигурация Django-проекта.

## Что это за репозиторий

`Library Project` — веб-приложение на Django для небольшого учебного или внутреннего библиотечного контура. Проект разделяет роли библиотекаря и читателя, хранит данные о книгах и выдачах, рассчитывает просрочки и позволяет выгружать отчёт по должникам.

## О чем проект

Проект покрывает базовые библиотечные процессы:

- каталог книг;
- реестр читателей;
- выдача и возврат книг;
- отслеживание просрочек;
- разделение интерфейсов по ролям (библиотекарь/читатель);
- экспорт отчёта по просроченным выдачам.

## Основные сущности

- `Book` — книга (`book_code`, название, автор, год, УДК);
- `Reader` — читатель (`reader_id`, ФИО, группа, связанный `User`);
- `Loan` — факт выдачи (дата выдачи, дата возврата по плану, фактический возврат).

В `Loan` реализованы бизнес-ограничения:
- дата выдачи не позже плановой даты возврата;
- фактический возврат не раньше даты выдачи;
- у одной книги не может быть более одной активной (не закрытой) выдачи.

## Пользовательские сценарии

### Библиотекарь

- Просмотр сводного дашборда (книги, активные выдачи, просрочки, суммарный долг);
- Создание читателя и учётной записи;
- Оформление выдачи книги;
- Приём возврата;
- Выгрузка отчёта по просрочкам.

### Читатель

- Вход в личный кабинет;
- Просмотр активных выдач;
- Просмотр просроченных книг и накопленного долга;
- Просмотр истории возвратов.

## Как запустить проект

1. Перейдите в каталог проекта:
   ```bash
   cd library_project
   ```
2. Примените миграции:
   ```bash
   python manage.py migrate
   ```
3. (Опционально) Создайте суперпользователя:
   ```bash
   python manage.py createsuperuser
   ```
4. Запустите сервер разработки:
   ```bash
   python manage.py runserver
   ```
5. Откройте приложение в браузере:
   - `http://127.0.0.1:8000/`
